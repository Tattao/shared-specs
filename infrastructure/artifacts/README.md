# Autonomous Delivery Artifacts

This directory stores task-level evidence for the Codex autonomous delivery MVP.

Each task should write:

- `summary.md`
- `changed-files.txt`
- `validation.md`
- `residual-risks.md`
- `handoff.md`

Artifacts are coordination evidence only. Product facts must be adopted back into `osmx` code, docs, tests, runbooks, or PR evidence before they become authoritative.
